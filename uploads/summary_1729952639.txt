This research paper proposes an automated task scheduling scheme for large-scale cloud computing systems that utilizes deep learning and reinforcement learning techniques. The scheme involves monitoring and predicting system parameters in real-time using deep learning technology, and dynamically adjusting task scheduling strategies according to the real-time system state and task characteristics using reinforcement learning algorithm. The research aims to optimize system resource utilization and task execution efficiency. The proposed scheme is validated through experiments and found to provide performance advantages compared to traditional methods. The research concludes that deep learning and reinforcement learning have potential applications in automatic task scheduling for large-scale cloud computing systems.
